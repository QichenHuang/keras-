# Keras使用指南
Keras是编写深度学习模型的一个高效工具，是一个高层次的神经网络API，能以TensorFlow、CNTK、Theano作为后端，详情介绍查看[官方网站](https://keras.io/)，
本指南展示Keras的一些基本用法，部分用法功能从源代码的角度进行分析。
## 环境
本指南提供的代码使用环境如下（其他版本的环境或许也可以，这里仅列作参考）
> tensorflow-gpu==1.14.0  
> Keras==2.3.0
